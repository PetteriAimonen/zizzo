Zizzo - Älykäs ohjelma.

Sääntöjen mukainen python-pääohjelma on zizzo.py. Lisäksi toinen python-ohjelma
on clitest.py, jota voi käyttää komentoriviltä Zizzon testaamiseen.

Itse koodi on kirjastona kansiossa lib. Ennen testien aloittamista on
suositeltavaa ajaa ohjelma kerran, esimerkiksi testisyötteellä. Python-tulkki
tallentaa bytecodeksi kääntämänsä lähdekoodin .pyc-päätteelle, jolloin ohjelma
käynnistyy nopeammin seuraavilla suorituskerroilla.

Toimii ainakin python 2.4:llä.
